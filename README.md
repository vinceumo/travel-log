# travel-log

## Project

This project is build using [Hugo](https://gohugo.io/).

Build:

```bash
hugo
```

Serve project:

```bash
hugo server
```

Serve project with draft:

```bash
hugo server -D
```

Build CSS

```bash
npm run gulp scssBuild
```

Watch CSS

```bash
npm run gulp scssWatch
```

Build JS

```bash
npm run gulp jsBuild
```

Watch JS

```bash
npm run gulp jsWatch
```
